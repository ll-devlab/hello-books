# Voice Spelling App: Hello Books!

This is a small web app that lets you spell words using voice input or by typing. It shows the word, plays its pronunciation, and fetches definitions.  

**Note:** This is a small experimental project created for learning purposes and to help a family member.  
The code was generated with the help of ChatGPT and is not a professional-grade application (yet). I will likely redo the project in the future if it proves useful. 

## Features

- Voice spelling using the browser's Speech Recognition API
- Type words manually
- Hear pronunciation using SpeechSynthesis
- Fetch word definitions from a free API
- Play buttons for online pronunciation

## How to Use

1. Open `index.html` in a modern browser (Chrome, Edge, or Firefox recommended)
2. **Tap the microphone button** to spell letters by voice
3. Type words manually if needed
4. Tap **Submit** to see the word and definition
5. Use the play buttons to hear pronunciation
6. Expand/collapse the **“Quick Guide”** section for instructions
